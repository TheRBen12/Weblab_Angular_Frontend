Hilfe

    Wie starte ich ein Experiment?

    1. Über den Eintrag “Experimente” in der Navigationsliste gelangen Sie zu allen Experimenten.
    2. Klicken Sie die entsprechende Experimentkarte an, oder betätigen Sie den Button “zu den Versuchsexperimenen”, um zu den Versuchsexperimenten zu gelnagen.
    3. Wählen Sie einen Versuch aus über den Button, “Versuch auswählen”.
    4. Starten Sie den Versuch.

    Einstellungen

    1. Über den Link "Einstellungen" gelangen Sie zu den Standardeinstellungen.
    2. Mithilfe der Einstellungen könneen Sie entscheiden, wie die Experiment-und Versuchsexperimente präsentiert werden.
    3. Schalten Sie die progressive Offenlegung ein, dann werden Sie Schrittweise durch die Auswahl der Experimente geführt. Damit werden Ihnen nicht alle Experimente und Versuche zur Auswahl angezeigt, sonern nur die, welche Sie bereits abgeschlossenen haben und das jeweils nächste freigeschalteene Experiment.
    4. Sie haben die Möglichkeit die progressive Offenlegung auf Ebene der Experimente sowie auf Ebene der Versuchsexperimente einzuschalten.

    Tipps

    1. Wenn Sie den Button in der unteren rechten Ecke betätigen, blenden Sie wertvolle Tipps ein.

    Experimente

    1. Wissen Sie nicht mehr weiter, dann wenden Sie sich an den Versuchsleiter
