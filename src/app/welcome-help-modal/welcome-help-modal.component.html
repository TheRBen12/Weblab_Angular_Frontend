Allgemeine Informationen
  • Unter "Experimente" finden Sie alle Experimente.
  • Haben Sie ein Experiment angecklickt, gelangen Sie zu den Versuchsexperimenten.
  • Wenn Sie einen Versuch ausgewählt haben, dann werden Sie zu der Versuchsbeschreibung weiter navigiert.
  • Lesen Sie die Versuchsbeschreibung aufmerksam durch und starten Sie anschliessend den Versuch.
  • Wenn Sie die Durchfühurung der Experimente abbrechen wollen, dann informieren Sie bitte den Versuchsleiter. Beenden Sie die Durchführung der Experiemnte nicht, bevor sie den Versuchsleiter informiert haben!
Einstellungen
  • Unter "Einstellungen können Sie Standardeinstellungen aktivieren oder deaktivieren. Die Einstellungen können Ihnen helfen, die Durchführung der Experimente zu beschleunigen."
  • In den Einstellungen können Sie festlegen, wie die Schnittstelle die Experimente und die Versuche visualisieren soll. Zudem können Sie Einstellungen zur aktiven Benutzerführung vornehmen.
Hilfestellungen
  • Unter “Hilfe” finden sie nützliche Infos und Hilfestellungen.
  • In den Hilfestellungen wird beschrieben, wie Sie innerhlab der Webseite navigieren, welche Einstellungsmöglichkeiten Sie haben und wie Sie die Durchführung der Experimente beschleunigen können.
Diese Informationen können Sie jederzeit einblenden, klicken Sie hierzu den Button in der rechten unteren Ecke an.
